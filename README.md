# Learning Probabilistic Ordinal Embeddings

This repository contains the **unofficial PyTorch implementation** of the paper ["Learning Probabilistic Ordinal Embeddings for Uncertainty-Aware Regression"](https://openaccess.thecvf.com/content/CVPR2021/papers/Li_Learning_Probabilistic_Ordinal_Embeddings_for_Uncertainty-Aware_Regression_CVPR_2021_paper.pdf) (CVPR 2021).

> **Obs:** The official implementation uses PyTorch 1.0.0 and Python 2.7. This implementation uses PyTorch 2.9.1 and Python 3.12.
